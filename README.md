# esp-butt

Bluetooth intimate hardware (sex toys, etc) controller powered by [buttplug.io](https://buttplug.io/).

## Status

This project is still in fairly early development, it has some quirks and rough edges but is generally functional. Feel free to contact me if you are interested in building your own or contributing to the project.
